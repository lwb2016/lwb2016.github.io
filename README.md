# Hexo博客框架的生成模板
## npm install hexo-cli -g
## hexo init blog
## cd blog
## npm install
## hexo server

# Quick Start

Create a new post

1
$ hexo new "My New Post"
More info: Writing

Run server

1
$ hexo server
More info: Server

Generate static files

1
$ hexo generate
More info: Generating

Deploy to remote sites

1
$ hexo deploy
